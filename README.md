# Pro Solutions Business Website

A modern and responsive business website built using React, Vite, and Tailwind CSS. This project showcases a clean and professional UI for a service-based company.

---

## 🚀 Live Demo
https://cwh-pro-solutions.vercel.app

---

## 📌 Features

- Fully responsive design (mobile, tablet, desktop)
- Modern landing page UI
- Service-based sections
- Clean and minimal design
- Fast performance with Vite
- Styled using Tailwind CSS

---

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

---

## 📂 Project Structure

```
├── src/
├── public/
├── index.html
├── package.json
├── vite.config.js
```

---

## 🎯 Purpose

This project was created to practice React and build a real-world business website interface with modern frontend tools.

---

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/srijaysahoo/pro-solutions-website

# Navigate to project folder
cd pro-solutions-website

# Install dependencies
npm install

# Run development server
npm run dev
```
